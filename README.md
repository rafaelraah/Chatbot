# 🤖 Chatbot com Memória usando Groq + Python

Este projeto é um chatbot inteligente desenvolvido em Python utilizando a API da Groq e modelos de linguagem avançados. O chatbot possui memória de conversa, permitindo manter contexto entre mensagens e criar interações mais naturais e dinâmicas.

## 🚀 Tecnologias utilizadas

* Python
* Groq API
* GPT OSS 120B
* Streamlit

## ✨ Funcionalidades

* Conversa em tempo real
* Memória de contexto
* Interface simples e moderna
* Integração com IA generativa
* Configuração segura com `.env`

## 📚 Objetivo do projeto

Este projeto foi criado com foco em aprendizado e prática de integração com APIs de Inteligência Artificial, explorando conceitos modernos como:

* Chatbots com memória
* Engenharia de prompts
* Integração com LLMs
* Estruturação de aplicações IA

## ▶️ Como executar

Instale as dependências:

```bash
pip install -r requirements.txt
```

Execute o projeto:

```bash
streamlit run app.py
```

## 🔐 Configuração da API

Crie um arquivo `.env`:

```env
GROQ_API_KEY=sua_api_key_aqui
```
